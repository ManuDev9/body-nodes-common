

Based on maven 
Testing based on JUnit

Usage:

mvn test

Modules:
  - BnAxisConfig: It is an utility object to easily change the axis values of your quaternions
  - BnConstants: It is a static object that contains all main constants of bodynode
  - BnReorientAxis: This is an utility object to easily reorient the axis of bodynodes data to the main
                    application in cases where the axis references are different 

Tested On:
  - 22.04.3-Ubuntu - openjdk 21.0.8 2025-07-15 - Apache Maven 3.8.7
